# ugh

Command-line **u**tilities for **G**it**H**ub

## Installation

```
npm i -g amo-ugh
```

## Usage

```
ugh <command>
```

## Commands

### `login`

Login to a GitHub account

### `release:cleanup <repository> <version> [--previous <previous>]`

Deprecate the previous branch lineage for the specified release version
